# msp430f2274

* Vcc: 2.8 - 3.6
* Wake up 0.1 us
* 16 bit RISC
* 16 registers
* Seven addressing modes
* 62.5 ns Cycle time
* Up to 16 MHz
* 16 bit timer A with 3 capture/compare
* 16 bit timer B with 3 capture/compare
* UART, SPI, I<sup>2</sup>
* 10 bit A/D Converter
* Two config opamps
* Bootstrap detector
* 32KB + 256 B Flash
* 1kb RAM
* Watchdot
* 2x8 I/O interrupt capable
